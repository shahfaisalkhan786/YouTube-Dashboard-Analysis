# YouTube Dashboard Analysis (Power BI)

## Project Overview
This project is an interactive Power BI dashboard that analyzes YouTube video performance across different categories.

## Dashboard Highlights
- Total Videos
- Total Views
- Total Earnings
- Total Comments
- Average Engagement
- Video Share by Category
- Earnings by Category
- Likes vs Comments
- Subscriber vs Views
- Likes & Earnings Comparison

## Tools Used
- Power BI Desktop
- Microsoft Excel / CSV
- DAX
- Power Query

## Key Insights
- Sports category generated the highest earnings.
- Music has the highest share of uploaded videos.
- Entertainment and Gaming contribute significantly to views.
- Subscriber growth varies across categories.
- Engagement differs by content category.
